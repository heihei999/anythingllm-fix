# AnythingLLM Desktop Patcher

修复 AnythingLLM 桌面版的已知 Bug。

## 下载

从 [Releases](https://github.com/heihei999/anythingllm-fix/releases) 页面下载最新版补丁包（zip），解压后双击 `patcher.bat` 即可。

## 前置条件

1. 已安装 [AnythingLLM 桌面版](https://anythingllm.com)
2. 已安装 [Node.js](https://nodejs.org/)（LTS 版本，安装后重启电脑）

## 使用方法

1. 关闭 AnythingLLM
2. 解压补丁包
3. 双击 `patcher.bat`
4. 看到"修补完成"后重新打开 AnythingLLM

详见补丁包内的 README。
